# Restaurant Management System

A desktop-based Restaurant Management System built using Java and JavaFX.  
The system is designed to manage restaurant operations including reservations, meal management, customer handling, and table organization.

## Features

- User authentication system
- Admin management panel
- Guest management
- Meal management
- Table reservation system
- Receptionist operations
- Restaurant table management
- File-based data storage
- Simple desktop interface using JavaFX

## System Roles

### Admin
- Manage users
- Manage meals
- View reservations
- Manage restaurant tables

### Receptionist
- Add guests
- Create reservations
- Assign tables
- View booking details

### Guest
- View available meals
- Make reservations
- View booking information

## Classes

The system is built using Object-Oriented Programming principles with the following main classes:

- Admin
- Guest
- Meals
- Receptionist
- Reservations
- RestaurantManagementSystem
- Table
- User
- Main

- NetBeans

3. Run application

bash Run Main.java 

## Data Storage

The system stores and retrieves data using Java File I/O.

Stored data:
- Users
- Meals
- Reservations
- Tables

Data is loaded into Lists/ArrayLists during runtime and saved back to files after updates.
